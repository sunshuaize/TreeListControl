# 自定义 TabControl 设计方案

## 1. 整体结构
- 从 `Control` 继承，完全自绘
- 包含两个区域：标签栏（顶部，高度 30px）+ 内容区域

## 2. 标签样式

### 默认状态
- 背景色：#F0F0F0
- 文字色：#323232
- 边框色：#CDCDCD

### 选中状态
- 背景色：#5D80F3
- 文字色：#FFFFFF
- 边框色：#F0F0F0

- 字体：微软雅黑 9pt
- 边框：1px
- 标签之间无间距，相邻标签共享边框（只显示外边框）

## 3. 布局
- 标签栏：顶部，高度 30px
- 内容区域：白色 #FFFFFF
- 标签支持文字显示

## 4. 核心功能
- 点击标签切换内容
- 支持添加/移除标签
- 从 Control 完全自绘，灵活控制外观
